# 🖥️ Basic Linux Commands for Beginners

## 📌 Introduction

Now that you have successfully installed Kali Linux, it’s time to learn some **basic Linux commands**.

These commands help you:

* Navigate through the system
* Manage files and folders
* Understand your environment

👉 Learning these basics is the first step to becoming comfortable with Linux.

---

## ⚙️ Basic Commands & Their Uses

### 1. `sudo su`

**Definition:**
This command gives you **root (administrator) access**.

👉 It allows you to perform tasks that require special permissions.

---

### 2. `ls`

**Definition:**
The `ls` command is used to **list files and folders** in the current directory.

👉 It helps you see what is inside a folder.

---

### 3. `pwd`

**Definition:**
The `pwd` command shows your **current working directory**.

**Stand for:**
👉 *Print Working Directory*

---

### 4. `cd .`

**Definition:**
This command refers to the **current directory**.

👉 It does not change anything but represents where you are.

---

### 5. `cd ..`

**Definition:**
This command moves you **one step back (parent directory)**.

👉 Useful when you want to go back to the previous folder.

---

### 6. `cd kali`

**Definition:**
This command moves you into a folder named **kali**.

👉 Used to navigate into a specific directory.

---

### 7. `cd Desktop`

**Definition:**
This command takes you to the **Desktop directory**.

👉 Helps you quickly access files on your Desktop.

---

### 8. `uname`

**Definition:**
The `uname` command shows **system information**.

👉 It tells you about your operating system.

---

### 9. `uname -r`

**Definition:**
This command shows the **kernel version** of your system.

👉 Useful for checking system details.

---

## 🎯 Final Note

Practice these commands daily to become comfortable with Linux.

👉 The more you practice, the easier it will become.
